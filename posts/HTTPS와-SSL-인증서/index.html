⭐2 [학습] HTTPS와 SSL 인증서 — Cloudflare가 다 해준다

By 코드대장간
Posted Mar 24 Updated Mar 26 3 min read ⭐2

모바일 앱에서 HTTP는 안 된다

TasteNote가 토스 미니앱이다. 모바일 앱에서 API를 호출하려면 HTTPS가 필수. iOS의 ATS(App Transport Security), Android의 Network Security Config 모두 HTTP 요청을 기본 차단한다. HTTPS가 없으면 API가 먹통.

HTTP vs HTTPS

HTTP는 평문 전송이다. 비밀번호를 적은 엽서를 우체부한테 맡기는 거다. 누구든 배달 중간에 들여다볼 수 있다. HTTPS는 봉투에 넣어서 봉인. TLS로 암호화한다.

 HTTPHTTPS
포트80443
암호화없음TLS
주소창“주의 요함”자물쇠
모바일 앱기본 차단정상

SSL 인증서

HTTPS를 쓰려면 SSL 인증서가 필요하다. 두 가지 역할.

  1. 신원 증명: “이 서버가 진짜 이 도메인이 맞다”는 보증
  2. 암호화 키 교환: 브라우저와 서버가 안전한 통신 채널을 만드는 데 사용

인증서를 받는 법

Let’s Encrypt (직접 설치)

무료. certbot으로 발급. 90일마다 갱신 필요. 서버에서 80포트로 도메인 인증을 받아야 한다. 직접 관리해야 할 게 있다.

Cloudflare (자동)

Cloudflare 프록시를 사용하면 인증서가 자동 발급·갱신된다. 서버에 certbot이나 인증서 설치가 필요 없다. Cloudflare Tunnel을 쓰고 있으니까 이 방식을 선택.

구간별 프로토콜

1

브라우저 ←HTTPS→ Cloudflare ←Tunnel→ cloudflared ←HTTP→ Nginx ←HTTP→ Spring Boot
구간프로토콜안전?
브라우저 ↔ CloudflareHTTPSO (TLS)
Cloudflare ↔ cloudflaredTunnelO (터널 자체가 암호화)
cloudflared ↔ NginxHTTP (localhost)O (같은 서버 내부)
Nginx ↔ Spring BootHTTP (localhost)O (같은 서버 내부)

서버 내부는 localhost 통신이라 HTTP여도 안전하다. 외부 구간만 암호화되면 된다. 집 안에서 가족끼리 하는 대화를 암호화할 필요 없는 거다.

SSL 모드

Cloudflare Dashboard에서 SSL/TLS 모드를 설정한다.

모드브라우저↔CFCF↔서버설명
OffHTTPHTTP암호화 없음
FlexibleHTTPSHTTPCF가 HTTPS 처리. 서버는 HTTP
FullHTTPSHTTPS서버에도 인증서 필요
Full (Strict)HTTPSHTTPS유효한 인증서 필요

Cloudflare Tunnel을 쓰면 터널 자체가 암호화돼 있으니까 Flexible로도 충분하다. 서버에 인증서를 설치할 필요 없다.

결론

Cloudflare Tunnel을 쓰면 HTTPS 관련해서 직접 할 일이 없다. 인증서 발급, 갱신, 설치 전부 자동. DNS와 Nginx 설정만 하면 끝.
